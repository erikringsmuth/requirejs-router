Try changing the values in the address bar and see them here.

Path Variables

The pathArg1 path variable is set in the demo route in main.js. You can inline the value like this "{{routeArguments.pathArg1}}".

Query Parameters

The queryParam1 query parameter is pulled from the URL. You don't have to specify query parameters in your routes. They're automatically added to the routeArguments. You can inline the value like this "{{routeArguments.queryParam1}}".
