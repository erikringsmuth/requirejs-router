Bower

Bower is a package manager for the web. Running bower install requirejs-router in the command line will install the router in the current directory. You can also set up a bower.json file in the root directory of your project.

{
  "name": "app-name",
  "dependencies": {
    "requirejs": "~2.1.10",
    "requirejs-text": "~2.0.10",
    "requirejs-router": "~0.7.3"
  }
}

In the command line switch to your root directory and run bower install. It will get all of the dependencies and put them in /bower_components.

Build

  • Clone git@github.com:erikringsmuth/requirejs-router.git
  • Run npm install to install dependencies
  • Run gulp to lint and minify your code. This will also watch for changes.

Running Tests

Open /tests/AmdSpecRunner.html and make sure all tests pass.
